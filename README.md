# Hasee K670D Hackintosh

## #电脑配置

- 型号：神舟K67D G4D1
- 操作系统：macOS Catalina 10.15.3
- 处理器：**8700es QN8H**
- 硬盘：**Samsung PM961**
- 显卡：英特尔 HD Graphics 630
- 声卡：Realtek ALC269VC (0x0269)
- 网卡：**Bcm94352z**

**加粗部分表示更换过配件**

BIOS更新至d大最新

BIOS中关闭CFG锁

核显使用的7代cpu核显驱动id，使用8代核显id发现不能亮度调节

此配置7代u同样适用，本子所用的模具相同即可

## #什么有效

- 核显
- 声音
- 触摸板（部分手势不能用 无解）
- USB端口
- LAN/以太网
- WIFI/蓝牙
- 睡眠（可正常唤醒）
- HDMI/HDMI音频
- 亮度调节
- 原生态电源管理/变频



## #什么无效
 - 开机插入hdmi，显示器亮屏，内建显示器黑屏，模式切换镜像显示，一切正常
